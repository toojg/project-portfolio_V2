CHANGE DIXIE

I was born and raised in southern Utah. It wasn't until I left home that I really started thinking about the relationship that its nickname, 'Dixie', had with the Antebellum South.

I started this project to spread information about the area’s unsavory connections to racism, as well as encourage southern Utahns to talk about changing Dixie State University's name.

The website exhibits newspapers, yearbooks, and articles that connect southern Utah to the Confederacy. I wanted the project to be firm, but not overbearing, and prompt the reader to think about these issues in an inviting way.

changedixie.com

This project was chosen by AIGA SLC to be displayed in The 100 Show (2020).
